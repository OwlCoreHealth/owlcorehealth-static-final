# Pasta Prodentim

Esta pasta contém arquivos JSON relacionados ao suplemento **Prodentim**. Cada arquivo descreve um sintoma específico e as fases de interação do chatbot para esse sintoma.

## Sintomas abordados na pasta:

- Mau hálito
- Inflamação nas gengivas
- Sensibilidade dentária
- Acúmulo de placa
- Cáries dentárias
- Gengivas sangrando
- Boca seca
- Úlceras na boca
- Acúmulo de tártaro
- Doenças periodontais
- Bruxismo (ranger de dentes)

Cada um desses sintomas tem seu próprio arquivo JSON correspondente, como:

- `prodentim-bad-breath.json`
- `prodentim-gum-inflammation.json`
- `prodentim-tooth-sensitivity.json`
- ... e assim por diante.

## Como Usar:

1. **Adicione novos arquivos JSON** para outros sintomas ou modificações existentes.
2. **Siga o formato padrão** para garantir consistência e funcionalidade no sistema.
3. **Validação**: Valide os arquivos JSON usando ferramentas como [JSONLint](https://jsonlint.com).

Esta pasta é essencial para o funcionamento do chatbot relacionado ao Prodentim.
